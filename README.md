# LEWS - Landslide Early Warning System

A real-time monitoring and alert system designed to help detect potential landslide risks using sensor-based data analysis and monitoring.

## Tech Stack

- HTML
- CSS
- JavaScript
- Python
- Django
- SQL

## Features

- Real-time monitoring dashboard
- Sensor activity visualization
- Early warning alerts
- Data-driven analysis system
- Responsive modern UI
